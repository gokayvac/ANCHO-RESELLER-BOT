##  Komutlar
- `/fiyatlar` - Ürün fiyatlarını görüntüle
- `/ürünlerim` - Satın alınan ürünleri listele
- `/lisanssil` - Lisans silme
- `/hwidreset` - HWID sıfırlama
- `/bakiye` - Bakiye görüntüleme ve yükleme
- `/admin` - Admin Komutları
